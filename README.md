> 사랑하는 사람들과 찍은 네컷사진들 폴더 여기저기 관리하기 힘드셨죠?
> 이제는 흩어진 네컷추억들을 쉽게 모아보세요

<br/>

1. 네컷 모아보기
  - 그리드 형식으로 한눈에 네컷사진을 확인할 수 있어요
  - 카드를 넘기면서 다양한 추억들을 확인할 수 있어요
2. 앨범커버 꾸미기
  - 나만의 네컷앨범커버를 꾸며보세요
3. 네컷사진 등록하기
  - 딱 필요한 정보만, 네컷사진과 찍은 날짜만 등록할 수 있어 간편해요
